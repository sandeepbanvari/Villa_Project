# 🏡 Villa Agency Website

A modern, fully responsive real estate website built using **HTML5, CSS3, and JavaScript**. The project provides an elegant user interface for showcasing luxury properties, property details, and contact information. It is designed with a clean layout, responsive design, and interactive navigation to deliver an excellent user experience across all devices.

---

## 📸 Preview

Villa Agency is a responsive multi-page real estate website that includes:

* Home Page
* Properties Listing Page
* Property Details Page
* Contact Page

---

## ✨ Features

* Fully Responsive Design (Desktop, Tablet & Mobile)
* Modern and Clean User Interface
* Sticky Navigation Bar
* Mobile Hamburger Menu
* Featured Property Section
* Property Listings with Detailed Information
* Property Details Page
* Contact Form
* Google Maps Integration
* Interactive Accordion Section
* Responsive Image Layouts
* Hover Effects and Smooth Transitions
* SEO-Friendly HTML Structure
* Reusable Components and Organized CSS

---

## 🛠️ Technologies Used

* **HTML5** – Semantic and accessible webpage structure
* **CSS3** – Responsive layouts, Flexbox, Grid, animations, and media queries
* **JavaScript (ES6)** – Mobile navigation and interactive UI functionality
* **Font Awesome** – Modern icons for navigation and content
